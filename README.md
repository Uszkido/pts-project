# 📱 Police/Phone Tracking System (PTS)

## Overview
The Phone Tracking System (PTS) is a platform designed to help users register, track, and verify mobile devices using IMEI or Serial Numbers. It aims to reduce phone theft and improve recovery through collaboration with authorities and stakeholders.

## Features
- 📌 IMEI/Serial Number Registration
- 🚨 Report Stolen or Lost Phones
- 🔍 Device Verification System
- 📍 Tracking & Monitoring
- 🔐 Secure User Authentication
- 🤝 Collaboration with Law Enforcement

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: PostgreSQL
